Technologies:

Code Front Code Back

J'ai développé à partir de zéro une « Application Web » de commerce électronique du type Amazon. Le site propose des produits de six catégories. Lorsque l'utilisateur accède au site Web, il a la possibilité de :

  • Consulter les produits affichés dans la page d’accueil.
  • Consulter les produits par catégorie.
  • Consulter les détails du produit.
  • Rechercher de produit par son nom.
  • Gérer le panier (ajout/ modification/ suppression).
  • Se connecter/ s'inscrire / se déconnecter (avec Okta)
  • Passer une commande (seulement s'il s’est identifié) et obtenir un numéro de suivi (qui est envoyé par le back-end).
